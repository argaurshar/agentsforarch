# Vendored skill: frontend-slides

The files in this directory are vendored **verbatim** from the open-source
`frontend-slides` skill:

- Source: https://github.com/zarazhangrui/frontend-slides
- Author: zarazhangrui

They are the skill's own instruction and resource files (`SKILL.md`,
`viewport-base.css`, `html-template.md`, `animation-patterns.md`,
`STYLE_PRESETS.md`). AND Studio's Concept Presentation feature feeds these
files to Claude so that every generated deck follows the skill precisely —
its fixed 1920×1080 stage, distinctive design system, and animation grammar.

Do not edit these files to "improve" them locally: keeping them verbatim is
what makes the generated output faithful to the upstream skill. To update the
skill, re-vendor the files from the source repository.
